JavaScript is a single threded multiparadigm programming language. It mainly runs on browser. JavaScript is a dynamic language. It is being used to make a website interactive. It is an intetpreted language. That means the interprator which is inside a browser reads the JavaScript code line by line and executes it. Because of this, JavaScript is a very fast language.

Event loop is a JavaScript concept. It is a way to run JavaScript code. We know that JavaScript is a single threaded programming language. That means it can run only one code each time. But if the code contains some functions, that takes some time to execute, then other code will do nothing but wait for the function to complete. That is why JavaScript has an event loop. It is a way to run the code in a loop. Event loop keeps functions in a stack and in a queue, and it executes the functions one by one. If the stack gets empty then it runs functions from the queue.

# LocalStorage SessionStorage
1 LocalStorage is a storage is the browser. SessionStorage is a storage is the browser.
2 LocalStorage is parmanent. SessionStorage is temporary.
3 LocalStorage does not get clear on browser close. SessionStorage gets clear on browser close.
4 LocalStorage does not get clear on tab close. SessionStorage gets clear on tab close.
4 LocalStorage does not get clear on tab change. SessionStorage gets clear on tab change.
